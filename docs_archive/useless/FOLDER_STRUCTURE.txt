TROUBLESHOOTING
================================================================================

PROBLEM: Can't find scripts
SOLUTION:
  - Look in: Quick Launch/ (root folder)
  - Not in Documentation/ folder

PROBLEM: Can't find documentation
SOLUTION:
  - Look in: Quick Launch/Documentation/
  - Not in root folder

PROBLEM: Script won't run
SOLUTION:
  1. Right-click the .bat file
  2. Select "Run as administrator"
  3. Try again

SOLUTION:
  1. Open: Quick Launch/Documentation/
  2. Read: README.md or GETTING_STARTED.txt
