# Mentor Dashboard - Quick Guide

## What's New?

### 1. Quick Actions (Top Right)
Three buttons now appear in the top right corner:
- **New Session** - Create a mentoring session
- **Start Mentoring** - Go to mentoring page
- **Messages** - View messages

### 2. Sessions Sidebar (Right Column)
A new sidebar on the right shows your recent sessions:
- Student name
- Session date
- Duration
- Rating (stars)
- Notes

### 3. Two-Column Layout
- **Left (8 columns)**: My Students table
- **Right (4 columns)**: Recent Sessions sidebar

## Troubleshooting

### Sessions not showing?
- Make sure you've created at least one session
- Refresh the page
- Check if backend is running

### Can't create session?
- Select a student from dropdown
- Fill in all required fields
- Check browser console for errors

### Sidebar not visible?
- Check screen size (should be visible on desktop)
- Scroll down on mobile
- Refresh the page
