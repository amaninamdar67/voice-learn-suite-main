TROUBLESHOOTING
================================================================================

Port Already in Use:
  1. Run: CHECK_SERVICES.bat
  2. Run: STOP_ALL.bat
  3. Try again

Node.js Not Found:
  1. Download from https://nodejs.org/
  2. Install with default settings
  3. Restart computer

Script Won't Run:
  1. Right-click .bat file
  2. Select "Run as administrator"

Services Won't Start:
  1. Run: CHECK_SERVICES.bat
  2. Run: STOP_ALL.bat
  3. Check for errors in terminal
  4. Try again

For More Help:
  See: README.md or GETTING_STARTED.txt

TIPS & TRICKS
================================================================================

TIP 1: Run as Administrator
  - Right-click .bat file → Run as administrator
  - Better compatibility

TIP 2: Keep Terminals Open
  - Scripts pause at end
  - Press any key to close
  - Or keep open to see logs

TIP 3: Watch Real-Time Logs
  - Each service has its own terminal
  - See errors as they happen

TIP 4: Use the Menu
  - MENU.bat is easiest
  - Interactive and user-friendly

TIP 5: Check Status Regularly
  - Run CHECK_SERVICES.bat anytime
  - Verify everything is running

TIP 6: Restart Single Service
  - Find its terminal window
  - Press Ctrl+C to stop
  - Run start script again
