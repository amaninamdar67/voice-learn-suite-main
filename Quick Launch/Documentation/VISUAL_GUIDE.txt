QUICK DECISION TREE
================================================================================

Are you a beginner?
  YES → Use MENU.bat (interactive menu)
  NO  → Use specific script

Do you know what you want?
  YES → Use specific script
  NO  → Use MENU.bat

Do you want to start backend only?
  YES → START_BACKEND_SERVERS.bat
  NO  → START_ALL.bat

Do you want to stop everything?
  YES → STOP_ALL.bat
  NO  → STOP_BACKEND_SERVERS.bat

Do you want to check status?
  YES → CHECK_SERVICES.bat
  NO  → Use other scripts

SCRIPT EXECUTION TIMELINE
================================================================================

START_BACKEND_SERVERS.bat:
  0s   - Script starts
  1s   - Check Node.js installation
  2s   - Start Main Backend (3001)
  4s   - Start Voice Server (8002)
  6s   - Start Whisper Server (8003)
  8s   - Start Ollama (11434)
  10s  - All servers running
  11s  - Display status
  12s  - Pause for user

STOP_BACKEND_SERVERS.bat:
  0s   - Script starts
  1s   - Kill port 3001
  2s   - Kill port 8002
  3s   - Kill port 8003
  4s   - Kill port 11434
  5s   - Kill all Node processes
  6s   - Display status
  7s   - Pause for user

START_ALL.bat:
  0s   - Script starts
  2s   - Start Main Backend (3001)
  4s   - Start Voice Server (8002)
  6s   - Start Whisper Server (8003)
  8s   - Start Ollama (11434)
  10s  - Start Frontend 1 (8000)
  13s  - Start Frontend 2 (8001)
  16s  - Start Frontend 3 (8002)
  19s  - All services running
  20s  - Display status
  21s  - Pause for user

================================================================================
COMMON ISSUES & SOLUTIONS
================================================================================

ISSUE: Port Already in Use
SOLUTION:
  1. Run: CHECK_SERVICES.bat
  2. Run: STOP_ALL.bat
  3. Try again

ISSUE: Node.js Not Found
SOLUTION:
  1. Install from https://nodejs.org/
  2. Restart computer
  3. Try again

ISSUE: Script Won't Run
SOLUTION:
  1. Right-click .bat file
  2. Select "Run as administrator"

ISSUE: Services Won't Start
SOLUTION:
  1. Run: CHECK_SERVICES.bat
  2. Run: STOP_ALL.bat
  3. Check for errors
  4. Try again
