AVAILABLE SCRIPTS
================================================================================

1. START_BACKEND_SERVERS.bat
   - Starts all 4 backend servers
   - Ports: 3001, 8002, 8003, 11434
   - Use when: You only need backend services
   - Time to start: ~5 seconds

2. STOP_BACKEND_SERVERS.bat
   - Stops all 4 backend servers
   - Kills processes on ports: 3001, 8002, 8003, 11434
   - Use when: Shutting down backend or restarting
   - Time to stop: ~3 seconds

3. START_ALL.bat
   - Starts backend (4 servers) + frontend (3 instances)
   - Backend ports: 3001, 8002, 8003, 11434
   - Frontend ports: 8000, 8001, 8002
   - Use when: Full system startup
   - Time to start: ~15 seconds

4. STOP_ALL.bat
   - Stops backend (4 servers) + frontend (5 instances)
   - Kills all processes on ports: 3001, 8002, 8003, 8000-8004, 11434
   - Use when: Complete system shutdown
   - Time to stop: ~5 seconds

5. CHECK_SERVICES.bat
   - Shows which services are currently running
   - Lists all ports and their status
   - Shows Node processes
   - Use when: Troubleshooting or checking status
   - Time to run: ~2 seconds

QUICK START GUIDE
================================================================================

OPTION 1: Backend Only (Recommended for Development)
  1. Double-click: START_BACKEND_SERVERS.bat
  2. In IDE/Terminal: npm run dev
  3. Access: http://localhost:5173 (or your dev port)
  4. To stop: Double-click STOP_BACKEND_SERVERS.bat

OPTION 2: Everything (Full System)
  1. Double-click: START_ALL.bat
  2. Wait for all windows to open
  3. Access: http://localhost:8000 (or 8001, 8002)
  4. To stop: Double-click STOP_ALL.bat

OPTION 3: Manual Control
  1. Open Command Prompt
  2. Navigate to project folder
  3. Run individual commands from BACKEND_COMMANDS.txt
  4. Press Ctrl+C to stop each service

================================================================================
TROUBLESHOOTING
================================================================================

PROBLEM: "Port already in use"
SOLUTION:
  1. Run: CHECK_SERVICES.bat
  2. Find the process using the port
  3. Run: STOP_ALL.bat
  4. Try again

PROBLEM: "Node.js is not installed"
SOLUTION:
  1. Download from https://nodejs.org/
  2. Install with default settings
  3. Restart computer
  4. Try again

PROBLEM: "Ollama not found"
SOLUTION:
  - This is optional, backend will work without it
  - If needed, download from https://ollama.ai/

PROBLEM: "Script won't run"
SOLUTION:
  1. Right-click the .bat file
  2. Select "Run as administrator"
  3. Or open Command Prompt as admin

PROBLEM: "Services won't start"
SOLUTION:
  1. Run: CHECK_SERVICES.bat
  2. Kill any existing processes: STOP_ALL.bat
  3. Check if ports are free
  4. Try starting again

TIPS & TRICKS
================================================================================

TIP 1: Run as Administrator
  - Right-click .bat file → Run as administrator
  - Better compatibility and fewer permission issues

TIP 2: Keep Terminal Open
  - Scripts pause at end, press any key to close
  - Or keep open to see logs

TIP 3: View Real-Time Logs
  - Each service opens in its own terminal
  - Watch logs as they happen

TIP 4: Restart Single Service
  - Find its terminal window
  - Press Ctrl+C to stop
  - Run start script again

TIP 5: Check Port Usage
  - netstat -ano | findstr :PORT_NUMBER
  - Shows which process is using a port

TIP 6: Kill Specific Process
  - taskkill /PID PROCESS_ID /F
  - Use PID from netstat output
